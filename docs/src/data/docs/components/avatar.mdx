## Components

The Avatar component consists of:

- `Avatar` - The root container
- `AvatarImage` - The image element
- `AvatarFallback` - The fallback element shown when the image fails to load

## API Reference

### Avatar

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `class` | `string` | `undefined` | Additional CSS classes |

### AvatarImage

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `src` | `string` | `undefined` | The image source |
| `alt` | `string` | `undefined` | Alternative text for the image |
| `class` | `string` | `undefined` | Additional CSS classes |

### AvatarFallback

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `class` | `string` | `undefined` | Additional CSS classes |
| `children` | `JSX.Element` | `undefined` | The fallback content |
