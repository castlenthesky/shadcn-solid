## API Reference

### Label

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `for` | `string` | `undefined` | The id of the element the label is associated with |
| `class` | `string` | `undefined` | Additional CSS classes |
| `children` | `JSX.Element` | `undefined` | The label text content |

The Label component extends the native HTML `<label>` element, so all standard label attributes and events are supported.
