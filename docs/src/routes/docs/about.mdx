## About

[shadcn-solid](https://shadcn-solid.com) is a port of shadcn/ui for Solid/SolidStart. It's maintained by [hngngn](https://twitter.com/hnggngnn).

## Credits

- [@shadcn](https://twitter.com/shadcn) - The brilliant mind behind the designs, methodology, and implementation of the original [shadcn/ui](https://ui.shadcn.com).
- [Kobalte UI](https://kobalte.dev) - The headless components that power this project.
- [Ark UI](https://ark-ui.com) - The `date-picker` component that power this project.
- [Corvu](https://corvu.dev) - The headless components that power this project.
- [cmdk-solid](https://cmdk-solid.vercel.app) - The `command` components that power this project.
- [solid-sonner](https://solid-sonner.vercel.app) - The `toast` components that power this project.
- [embla-carousel](https://www.embla-carousel.com) - The `carousel` components that power this project.
- [Radix UI](https://www.radix-ui.com) - The headless components and examples that the original [shadcn/ui](https://ui.shadcn.com) was built on.

- [Shu Ding](https://shud.in) - The typography style is adapted from his work on Nextra.
