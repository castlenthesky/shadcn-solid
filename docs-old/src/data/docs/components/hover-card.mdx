## Usage

```tsx

```

```tsx
<HoverCard>
  <HoverCardTrigger>Hover</HoverCardTrigger>
  <HoverCardContent>
    SolidJS - Simple and performant reactivity for building user interfaces.
  </HoverCardContent>
</HoverCard>
```
